    ## Pair it with AstroCMS

    This template ships with [AstroCMS](https://github.com/lonestone/astrocms) pre-configured. Editors update content through a web UI and the CMS commits the MDX changes straight to Git. No database, no backend, no risk of divergence between staging and production.

    Spin it up with one command: `npm run astrocms`.
  </Fragment>

    ## FAQ

    *A few quick answers*

    Can't find what you're looking for? [Get in touch](/en/contact).
  </Fragment>

  <Fragment slot="right">
    <FaqItem question="Is this template free to use?">
      Yes. It is released under the MIT license. Use it as-is for personal or commercial projects.
    </FaqItem>

    <FaqItem question="Do I have to keep every section?">
      No. Every collection (blog, case studies, docs, etc.) is optional. Delete the ones you don't need and the rest keeps working.
    </FaqItem>

    <FaqItem question="How do I add a new language?">
      Add the locale to `website.config.ts`, create a translation file in `src/content/translations/<lang>.yaml`, and start adding content files suffixed with `/<lang>.mdx`.
    </FaqItem>

    <FaqItem question="Does AstroCMS require a backend?">
      No. [AstroCMS](https://github.com/lonestone/astrocms) edits the MDX files directly and can commit changes back to Git. Nothing else is needed.
    </FaqItem>
  </Fragment>

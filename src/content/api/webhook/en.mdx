## Fields

| Field    | Type        | Description                                          |
| -------- | ----------- | ---------------------------------------------------- |
| `id`     | `uuid`      | Unique identifier for the webhook                    |
| `url`    | `String`    | Destination URL that receives the POST requests      |
| `events` | `String[]`  | Event names this webhook listens to                  |
| `active` | `Boolean`   | Whether the webhook is currently delivering events   |

## Delivery

On each matching event, the product sends a `POST` request to `url` with a JSON body:

```json
{
  "event": "user.created",
  "payload": { /* entity snapshot */ },
  "deliveredAt": "2026-04-01T10:15:00Z"
}
```

<Callout type="info" title="Retries">
  Failed deliveries are retried up to 5 times with exponential backoff.
</Callout>

## Delivery

On each matching event, the product sends a `POST` request to `url` with a JSON body:

```json
{
  "event": "user.created",
  "payload": { /* entity snapshot */ },
  "deliveredAt": "2026-04-01T10:15:00Z"
}
```

<Callout type="info" title="Retries">
  Failed deliveries are retried up to 5 times with exponential backoff.
</Callout>

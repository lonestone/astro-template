---
title: User
description: Représente un utilisateur de votre compte. Créé automatiquement à la première connexion.
entity: user
category: core
---

## Champs

| Champ       | Type       | Description                                        |
| ----------- | ---------- | -------------------------------------------------- |
| `id`        | `uuid`     | Identifiant unique de l'utilisateur                |
| `email`     | `String`   | Adresse email principale, utilisée pour la connexion |
| `name`      | `String`   | Nom d'affichage                                    |
| `createdAt` | `DateTime` | Date de création du compte                         |

## Exemple de requête

```bash
curl https://api.example.com/v1/users/me \
  -H "Authorization: Bearer VOTRE_CLE_API"
```

Exemple de réponse :

```json
{
  "id": "d290f1ee-6c54-4b01-90e6-d701748f0851",
  "email": "ada@example.com",
  "name": "Ada Lovelace",
  "createdAt": "2026-04-01T10:15:00Z"
}
```
